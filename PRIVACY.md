# Privacy Policy - Bot Suport

Bot Suport only uses Discord data required to function, such as user ID, role ID, voice channel ID, guild ID, and voice state updates.

The bot does not collect personal messages, passwords, emails, or sensitive personal information.

The bot uses this data only to check staff permissions, create temporary support voice channels, move users into those channels, and delete empty support channels.

No data is sold or shared with third parties.
